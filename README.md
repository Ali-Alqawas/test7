**منصة ذكية متكاملة لإدارة العروض والخصومات التجارية**

---

## 📋 نظرة عامة

SIDE هو تطبيق Flutter متقدم يوفر تجربة تسوق إلكتروني متكاملة من خلال منصة موحدة لإدارة العروض والخصومات التجارية.

### 🎯 الهدف

- **التجميع الذكي**: عرض جميع العروض من مختلف المتاجر في مكان واحد
- **التخصيص**: نظام توصيات ذكي مبني على اهتمامات المستخدم
- **التفاعل**: تواصل مباشر مع التجار ونظام تقييمات شامل
- **المكافآت**: نظام نقاط ومكافآت لتحفيز المستخدمين

## ✨ المميزات المنجزة

### ✅ نظام المصادقة الكامل
- تسجيل الدخول والتسجيل
- التحقق عبر OTP
- استعادة كلمة المرور
- إدارة JWT Tokens
- التخزين الآمن للبيانات

### ✅ الشاشات الرئيسية
- **الشاشة الرئيسية**: عرض ديناميكي للعروض والفئات
- **شاشة العروض**: عروض حصرية، مجمعة، ومميزة
- **شاشة الفئات**: تصفح حسب الفئات
- **شاشة البحث**: بحث متقدم مع فلاتر
- **شاشة المفضلة**: حفظ العروض المفضلة
- **شاشة الملف الشخصي**: إدارة الحساب والإعدادات

### ✅ شاشات التفاصيل
- تفاصيل العرض الكاملة
- ملف التاجر
- التعليقات والتقييمات
- المحادثة مع التاجر

### ✅ شاشات الملف الشخصي
- تعديل الملف الشخصي
- المكافآت والنقاط
- السحوبات
- صندوق الوارد
- إعدادات الإشعارات
- الأمان والخصوصية
- العناوين المحفوظة
- مركز الدعم
- الترقية لحساب تاجر

### ✅ المكونات المخصصة
- نظام تصميم موحد (Design System)
- أزرار مخصصة
- حقول إدخال مخصصة
- بطاقات العروض
- شريط التنقل السفلي
- Glass Morphism Effects
- فلاتر العروض
- زر المساعد الذكي العائم

### ✅ البنية التقنية
- **إدارة الحالة**: Provider
- **الشبكة**: HTTP مع إدارة متقدمة للـ API
- **التخزين المحلي**: SharedPreferences
- **نظام الثيمات**: Dark/Light Mode
- **معالجة الأخطاء**: نظام استثناءات متكامل
